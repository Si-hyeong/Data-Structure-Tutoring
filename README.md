# Data-Structure-Tutoring

Practice exam quizzes for CSE203: Data Structures at DGIST.
Each quiz covers the weekly lecture material with interactive questions and Korean explanations.

---

## ✨ Features

- **5 question types** — True/False, Multiple Choice, Fill-in-the-blank, Programming, Draw
- **20 questions per week** — medium to hard difficulty
- **Interactive** — click options to select, toggle answers to reveal explanations
- **Filter by type** — focus on specific question categories
- **Korean explanations** with key terms in English

---

## 🛠 Built With

Plain HTML + CSS + JavaScript — no dependencies, no build step.
